Detector Motor

Motor de ajedrez desarrollado en C++ que implementa los algoritmos clásicos de búsqueda Minimax con Alpha-Beta pruning, junto con optimizaciones como ordenación de movimientos, búsqueda iterativa y una función de evaluación heurística multicomponente. Incluye una interfaz gráfica en Python con Pygame para visualizar el tablero, interactuar con el usuario y enviar comandos al motor.
